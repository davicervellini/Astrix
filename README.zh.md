# Astrix (v0.3)

**Astrix** (源自 Astra + Matrix) 是一种为人造的星际时代而优化的构造语言（人造语）。

## 宗旨
作为人类探索太空的官方辅助语言，致力于实现科学精确性、指令的清晰度以及与人工智能的无缝融合。

## 原则
* 纯粹的语音体系
* 规则的语法（无例外，无歧义）
* 高度符合逻辑
* 易于学习
* 具备极强的扩展性，专为人类与AI的共同优化而设计

## 基本规则

### 语音与拼写
* **精简字母表**: 仅使用发音固定、不含糊的字母：`p, t, k, b, d, g, m, n, f, v, s, z, l, r, j, y, a, e, i, o, u`。不使用容易引起混淆的字母，如 'c'、'q' 和 'x'。
* **元音**: a, e, i, o, u (发音永远保持一致).
* 简单的音节结构 (CV/CVC)。
* **拼字法**: 完全依照读音拼写（音标即拼写）。
* **重音**: 重音**永远**落在倒数第二个音节上。

### 词法（词尾后缀）
单词的最后一个元音毫无歧义地定义了该词的词性：
* **动词**: 以 `-a` 结尾 (例如：*parla* 说，*krea* 创造，*protekta* 保护).
* **名词**: 以 `-o` 结尾 (例如：*stello* 星星，*plano* 行星，*vako* 太空). 无语法性别。
* **形容词**: 以 `-i` 结尾 (放在名词之前，例如：*grandi* 大的，*boni* 好的).
* **副词和感叹词**: 以 `-u` 结尾 (例如：*salu* 你好，*vayu* 再见).
* **代词**: 基础元音为 `i` (例如：*mi* 我，*ti* 你).

### 语法
* **语序**: 主-谓-宾 (SVO).
* **时态** (放在动词之前的独立助词):
  * `nu` (现在时)
  * `pa` (过去时)
  * `fu` (将来时)
* **复数**: 在名词末尾加 `-s` (例如：*stellos* 星星们).

### 人称代词
* `mi` (我)
* `ti` (你)
* `li` (他/她/它 - 中性)
* `ni` (我们)
* `vi` (你们)
* `si` (他们/她们)

## 派生系统 (Derivation System)
Astrix 使用高度符合逻辑的派生系统，通过少量词根创造出大量新词。

### 元音转换 (Vowel Transmutation)
任何词根只需改变最后一个元音，即可改变其词性：
* `rapid-` (速度的词根)
* `rapidi` (快的 - 形容词)
* `rapidu` (快速地 - 副词)
* `rapida` (加速 - 动词)
* `rapido` (速度 - 名词)

### 前缀 (状态修饰符)
* `no-` (逻辑反义): `boni` (好) → `noboni` (坏)
* `max-` (增强/放大): `grandi` (大) → `maxgrandi` (巨大)
* `min-` (减弱/缩小): `grandi` (大) → `mingrandi` (小)

### 后缀 (名词生成器)
* `-ist-` (执行者/职业): `skopa` (探索) → `skopisto` (探索者)
* `-il-` (工具/机器): `parla` (说话) → `parlilo` (通讯器/无线电)
* `-ik-` (科学/学科): `stello` (星星) → `stelliko` (天文学)

## 句子示例

* **Mi fu parla Astrix kun ti.** → 我将和你一起说 Astrix 语。
* **Ni fu skopa stellos kun ti.** → 我们将一起探索群星。
* **Mi nu protekta ti, partnoro.** → 我保护你，搭档。
* **No tima vako.** → 不要害怕太空。
